Class: ElementGraphique

ElementGraphique()

La classe ElementGraphique est la base de tous les éléments de l'éditeur d'algorithmes .
Chaque Element Descent de ElementGraphique Lorsque la valeur par défaut est ElementGraphique chacun des types héritant de celui-ci peuvent être utilisé

new ElementGraphique()

Est utilisé pour instancier les héritier de la classe ElementGraphique.
Par défaut la position à la création de l'instance est à (0,0)
