selectTool(idTool)

Cette méthode met à jour l'outil actuellement sélectionné et modifie le curseur du plan principal et des sous-plans de travail en fonction de l'outil sélectionné. Si un outil autre que le lien est sélectionné, elle enlève la classe "pointePourLien" de l'élément précédemment pointé.
Parameters:
Name Type Description
idTool number L'identifiant de l'outil à sélectionner. Si idTool est -1, le pointeur est sélectionné.

(static) paste()

Cette fonction lit les données du presse-papiers, les parse en JSON, puis applique un décalage aux coordonnées de chaque élément basé sur la position actuelle de la souris. Si les éléments contiennent des enfants ou des conditions, le décalage est également appliqué récursivement à ces sous-éléments. Enfin, les données modifiées sont chargées dans l'éditeur.
Source:
  • PartieEditeur/Editeur.js, line 1772
Throws:
Lance une erreur si les données du presse-papiers ne sont pas au format JSON.
Type
Error
Example
// Exemple d'utilisation
editeur.paste();

Cette fonction lit les données du presse-papiers, les parse en JSON, puis applique un décalage aux coordonnées de chaque élément basé sur la position actuelle de la souris. Si les éléments contiennent des enfants ou des conditions, le décalage est également appliqué récursivement à ces sous-éléments. Enfin, les données modifiées sont chargées dans l'éditeur.
