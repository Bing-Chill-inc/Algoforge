Ce modèle permet de saisir une valeur (entier, caractère...) vérifiant une propriété donnée.

Si la valeur saisie ne respecte pas la propriété attendue, le modèle invite l'utilisateur à renouveler sa saisie jusqu'à ce qu'il saisisse une valeur correcte.

Exemple de cas d'usage : saisir une note dont la valeur est comprise entre 0 et 20.
